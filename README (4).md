# URL Shortener API

API-сервис сокращения ссылок на FastAPI.

Функциональность:
- создание короткой ссылки
- редирект по короткому коду на оригинальный URL
- обновление и удаление ссылки владельцем
- статистика по ссылке
- кастомный alias
- поиск по оригинальному URL
- время жизни ссылки через `expires_at`
- регистрация и логин пользователей
- история истекших и удалённых ссылок
- автоматическая очистка истекших и неиспользуемых ссылок
- кэширование через Redis

Используемый стек:
- FastAPI
- PostgreSQL
- Redis
- Celery
- SQLAlchemy
- Alembic
- Docker Compose

## Запуск

1. Запустить сервисы:

```bash
docker compose up --build
```

2. После запуска сервис будет доступен по адресам:
- `http://localhost:8000`
- `http://localhost:8000/docs`

3. Остановка:

```bash
docker compose down
```
